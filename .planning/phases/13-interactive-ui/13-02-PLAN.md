# Phase 13.2: Main Window + Dashboard

**Goal**: Add a SwiftUI main window with sidebar navigation (Dashboard, Search, Settings) and a Dashboard view showing sync history, recent notes, and classification stats.

## User Review Required

> [!IMPORTANT]
> This introduces the app's first visible window. Currently, the app is menu-bar-only with an invisible Settings window. After this phase, clicking "Open Dashboard" in the menu bar opens a full window.

> [!NOTE]
> The existing `SettingsView` moves from a standalone Settings scene into a sidebar tab within the main window.

## Proposed Changes

### 1. App Entry Point

#### [MODIFY] [SecondBrainApp.swift](file:///Users/richardyu/PARA/01_Projects/Personal/apps/second-brain/apple/SecondBrain/Sources/SecondBrain/SecondBrainApp.swift)
- Replace standalone `Settings` scene with a `Window` scene containing `MainWindowView`
- Initialize `AppDatabase.makeDefault()` in `AppDelegate` and pass to `MenuBarManager`
- Use `@Environment(\.openWindow)` to allow menu bar to open the window

### 2. Main Window Layout

#### [NEW] `Sources/SecondBrain/UI/MainWindowView.swift`
- `NavigationSplitView` with a 3-item sidebar:
  - **Dashboard** (icon: `chart.bar.fill`) — default selection
  - **Search** (icon: `magnifyingglass`) — placeholder for Phase 13.4
  - **Settings** (icon: `gear`) — embeds existing `SettingsView`
- Minimum window size: 700×500
- Enum `SidebarItem: String, CaseIterable` for the tabs

### 3. Dashboard View

#### [NEW] `Sources/SecondBrain/UI/DashboardView.swift`
- Accepts `AppDatabase` via init
- **Stats cards** (top row, HStack):
  - Total notes count
  - Notes synced today
  - Most common category (with icon)
  - Classification accuracy (% not corrected)
- **Recent syncs table** (middle): List of last 10 `SyncEvent` rows with timestamp, count, status badge (green/red)
- **Recent notes list** (bottom): Last 20 `Note` rows with title, category pill, destination path, and timestamp

#### [NEW] `Sources/SecondBrain/ViewModels/DashboardViewModel.swift`
- `@Observable` class holding dashboard state
- Loads data from `AppDatabase` on appear and after each sync
- Properties: `totalNotes`, `todayNotes`, `topCategory`, `accuracy`, `recentSyncs`, `recentNotes`

### 4. Menu Bar Wiring

#### [MODIFY] [MenuBarManager.swift](file:///Users/richardyu/PARA/01_Projects/Personal/apps/second-brain/apple/SecondBrain/Sources/SecondBrain/UI/MenuBarManager.swift)
- Replace "Settings..." with "Open Dashboard" (⌘D)
- Store reference to `AppDatabase`; pass to `NoteProcessor` on sync
- On sync complete, post `Notification` so dashboard can refresh

### 5. Database in AppDelegate

#### [MODIFY] [SecondBrainApp.swift](file:///Users/richardyu/PARA/01_Projects/Personal/apps/second-brain/apple/SecondBrain/Sources/SecondBrain/SecondBrainApp.swift)
- `AppDelegate` creates `AppDatabase.makeDefault()` on launch
- Passes database to both `MenuBarManager` and `MainWindowView` via environment
