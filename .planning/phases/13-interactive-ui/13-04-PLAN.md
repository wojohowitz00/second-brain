# Phase 13.4: Config Wizard + Search

**Goal**: Add a first-run setup wizard guiding new users through configuration and a full-text search view using the existing FTS5 index.

## User Review Required

> [!NOTE]
> The wizard only appears on first launch (when vault path is unset). Users can always reconfigure via Settings.

## Proposed Changes

### 1. Setup Wizard

#### [NEW] `Sources/SecondBrain/UI/WizardView.swift`
- Multi-step wizard using `TabView` with explicit selection control
- **Step 1 — Ollama**: Check Ollama reachability, show status indicator
- **Step 2 — Model**: Pick from available models (auto-fetched)
- **Step 3 — Vault**: Browse to Obsidian vault folder
- **Step 4 — Slack**: Enter token + channel ID, validate non-empty
- **Step 5 — Done**: Summary + "Get Started" button
- Uses existing `SettingsViewModel` for persistence (Keychain + UserDefaults)
- Navigation: Back/Next buttons with validation before advancing

#### [MODIFY] [SecondBrainApp.swift](file:///Users/richardyu/PARA/01_Projects/Personal/apps/second-brain/apple/SecondBrain/Sources/SecondBrain/SecondBrainApp.swift)
- In the Window scene body, check `UserDefaults.vaultPath.isEmpty`
- If empty → show `WizardView` instead of `MainWindowView`
- On wizard completion, transition to `MainWindowView`

### 2. Search View

#### [NEW] `Sources/SecondBrain/UI/SearchView.swift`
- Replaces `SearchPlaceholderView` in the sidebar
- **Search field** at top with debounced input (300ms)
- **Results list** showing matched notes with highlighted query terms
- Each result row: title, category pill, original text snippet, destination path
- Uses `AppDatabase.searchNotes(query:)` (FTS5)
- Empty state when no query or no results

#### [MODIFY] [MainWindowView.swift](file:///Users/richardyu/PARA/01_Projects/Personal/apps/second-brain/apple/SecondBrain/Sources/SecondBrain/UI/MainWindowView.swift)
- Replace `SearchPlaceholderView()` with `SearchView(database: database)`

### 3. Tests

#### [NEW] `Tests/SecondBrainTests/UI/SearchViewModelTests.swift`
- Test search returns FTS5 results
- Test empty query returns empty array
- Test no-match query returns empty array
