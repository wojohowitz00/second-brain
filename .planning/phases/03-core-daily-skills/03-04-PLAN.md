# Tasks by Status

Live dashboard — queries render automatically in Obsidian via Dataview.

---

## Overdue

\```dataview
TABLE without id
  file.link AS Task,
  priority AS Priority,
  due_date AS Due,
  project AS Project
FROM ""
WHERE type = "task"
  AND status != "done"
  AND due_date < date(today)
SORT priority DESC, due_date ASC
\```

---

## Due This Week

\```dataview
TABLE without id
  file.link AS Task,
  priority AS Priority,
  due_date AS Due,
  project AS Project,
  status AS Status
FROM ""
WHERE type = "task"
  AND status != "done"
  AND due_date >= date(today)
  AND due_date <= date(today) + dur(7 days)
SORT priority DESC, due_date ASC
\```

---

## All Tasks by Status

\```dataview
TABLE without id
  file.link AS Task,
  priority AS Priority,
  due_date AS Due,
  project AS Project,
  domain AS Domain
FROM ""
WHERE type = "task"
SORT priority DESC, due_date ASC
GROUP BY status
\```

---

- `TABLE without id` removes the default file column (we show file.link AS Task instead)

**Dashboard content:**

```markdown

type: dashboard
created: YYYY-MM-DD

# Project Health

Live dashboard — queries render automatically in Obsidian via Dataview.

---

## Active Projects

\```dataview
TABLE without id
  file.link AS Project,
  health AS Health,
  next_action AS "Next Action",
  deadline AS Deadline,
  domain AS Domain
FROM ""
WHERE type = "project" AND status = "active"
SORT health ASC, deadline ASC
\```

---

## Stale Projects (No Activity in 7+ Days)

\```dataview
TABLE without id
  file.link AS Project,
  health AS Health,
  file.mtime AS "Last Modified"
FROM ""
WHERE type = "project"
  AND status = "active"
  AND file.mtime < date(today) - dur(7 days)
SORT file.mtime ASC
\```

---

## All Projects

\```dataview
TABLE without id
  file.link AS Project,
  status AS Status,
  health AS Health,
  deadline AS Deadline,
  next_action AS "Next Action"
FROM ""
WHERE type = "project"
SORT status ASC, deadline ASC
\```

---

*Live Dataview dashboard — auto-refreshed when opened in Obsidian.*
*"At risk" = health: red (overdue tasks) or appears in Stale Projects (no activity 7+ days).*
```
